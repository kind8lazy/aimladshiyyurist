# AI Marketing Team для ЮрШтаб AI

Виртуальная маркетинговая команда из 7 AI-агентов под руководством CMO.

## Структура команды

```
CMO Agent (Orchestrator)
├── Brand Strategist
├── Market Researcher
├── Content Manager
├── Growth Marketer
├── Product Marketer
└── Social Media Manager
```

## Быстрый старт

```bash
# Перейти в папку агентов
cd agents

# Запустить CMO агента с задачей
node cli.js cmo "Разработай GTM стратегию"

# Запустить конкретного агента
node cli.js content_manager "Создай контент-план на 90 дней"
node cli.js market_researcher "Проведи конкурентный анализ"
node cli.js growth_marketer "Настрой лидогенерацию"
```

## Доступные агенты

| Агент | Описание | Пример задачи |
|-------|----------|---------------|
| `cmo` | Главный маркетолог-координатор | "Разработай стратегию запуска" |
| `brand_strategist` | Бренд и позиционирование | "Создай tone of voice" |
| `market_researcher` | Исследования рынка | "Проведи конкурентный анализ" |
| `content_manager` | Контент-маркетинг | "Напиши контент-план" |
| `growth_marketer` | Лидогенерация | "Настрой воронку" |
| `product_marketer` | Продуктовый маркетинг | "Подготовь pitch deck" |
| `social_media_manager` | Соцсети | "Создай контент для Telegram" |

## Архитектура

- `index.js` — главный модуль с конфигурацией агентов
- `cli.js` — CLI интерфейс для запуска
- `{agent}/SKILL.md` — описание роли и примеры
- `{agent}/agents/openai.yaml` — конфигурация для OpenAI Agents

## Интеграция с LLM

Текущая версия использует симуляцию ответов. Для реальной интеграции с LLM:

1. Добавьте API ключ в `.env`
2. Модифицируйте `cli.js` для вызова LLM
3. Передавайте `agent.prompt` как system prompt

## Workflow: Как агенты работают вместе

1. **CEO** отправляет задачу **CMO**
2. **CMO** анализирует и декомпозирует задачу
3. **CMO** делегирует подзадачи соответствующим субагентам
4. **Субагенты** выполняют свои задачи параллельно
5. **CMO** собирает результаты и синтезирует решение
6. **CMO** отчитывается перед **CEO**

## Разработка

Добавить нового агента:
1. Создать папку `agents/{new_agent}/`
2. Добавить `SKILL.md` с описанием роли
3. Добавить `agents/openai.yaml` с конфигом
4. Добавить агента в `index.js` → `AGENTS`
