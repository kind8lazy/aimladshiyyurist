### Screen Recording Guidelines

**Software**:
- Mac: QuickTime or ScreenFlow
- Windows: OBS Studio or Camtasia
- Resolution: Record at 2x (3840x2160) and downscale to 1080p for crisp UI

**Cursor**:
- Enable cursor highlighting (yellow circle on click)
- Slow down cursor movements (1.5x slower than normal)
- Pause cursor after clicks (0.5s) for clarity

**UI Prep**:
- Use demo data (no real client names)
- Clean browser: no bookmarks bar, close other tabs
- Disable notifications
- Use large font sizes (16px minimum for body text)

**Animations**:
- Progress bars: show realistic timing (don't skip)
- Loading states: keep visible (builds anticipation)
- Transitions: max 0.3s fade/slide

---

### Visual Style Guide

**Color Grading**:
- Slightly increase saturation (+10%) for screen recordings
- Boost contrast for text legibility
- Color code:
  - Risks = Red (#EF4444)
  - Deadlines = Yellow (#F59E0B)
  - Completed = Green (#10B981)

**Typography Overlays**:
- Font: Inter or Manrope (clean sans-serif)
- Title size: 72px
- Body size: 48px
- Animation: Fade in + slight slide up (0.3s)

**Iconography**:
- Use consistent icon set (e.g., Heroicons, Lucide)
- Animate icons with scale or fade
- Keep icon color consistent with brand

---

### Audio Production

**Voice-Over Recording**:
- Microphone: Condenser mic (e.g., Blue Yeti, Rode NT-USB)
- Environment: Quiet room, minimal echo
- Distance: 6-8 inches from mic
- Format: WAV 48kHz 24-bit
- Record 2-3 takes, pick best

**Audio Post-Processing**:
1. Noise reduction (gentle, -12dB)
2. EQ: High-pass filter at 80Hz, boost 3-5kHz for clarity
3. Compression: 3:1 ratio, -6dB threshold
4. Normalize to -3dB peak
5. Export: AAC 256kbps or WAV

**Music**:
- Style: Modern corporate, upbeat but not distracting
- BPM: 100-120
- Volume: -24dB to -18dB (low, background only)
- Sources: Epidemic Sound, Artlist, Musicbed
- Fade in/out: 2 seconds

**SFX**:
- UI clicks: Subtle, high-quality (not stock)
- Success sounds: For completions (upload done, analysis complete)
- Whoosh transitions: For scene changes
- Volume: -18dB to -12dB
