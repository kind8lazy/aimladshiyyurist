# IP Track: название и правовая стратегия бренда

## 1) Базовая архитектура

- Основной ТЗ для регистрации: `ЮрШтаб AI` (master-brand).
- `Младший Юрист AI 24/7` использовать как обозначение продуктовой персоны/сервиса.

## 2) Предварительная проверка (до подачи)

1. Поиск совпадений/схожести в базах ФИПС/Роспатента.
2. Проверка конфликтов по релевантным классам МКТУ.
3. Проверка коммерческого использования схожих имен в digital-каналах.
4. Оценка доменных и соцсетевых дескрипторов под master-brand.

## 3) Стратегия подачи

1. Подать `ЮрШтаб AI` как основной словесный/комбинированный ТЗ.
2. Для `Младший Юрист AI 24/7` подготовить fallback:
  - использовать как слоган/наименование функционала;
  - не делать единственным защищаемым именем компании.
3. Зафиксировать правила бренд-употребления в маркетинге и UI.

## 4) Правила claims в рекламе и продукте

- Запрещенные формулировки:
  - `гарантируем выигрыш`;
  - `полностью заменяем юристов`;
  - `100% защита от рисков`.
- Допустимые формулировки:
  - `вспомогательный инструмент legal operations`;
  - `ускоряет рутину и контроль сроков`;
  - `финальные решения принимает ответственный юрист`.

## 5) Операционный чек-лист

1. Согласовать финальные бренд-неймы с юристом по ИС.
2. Подготовить комплект документов для заявки ТЗ.
3. Обновить маркетинг-материалы и UI в соответствии с brand rules.
4. Встроить единый дисклеймер во все каналы продаж.
